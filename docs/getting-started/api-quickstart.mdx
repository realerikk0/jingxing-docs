# API 快速开始

井陉优先提供 OpenAI 兼容调用方式，便于现有 SDK 与客户端平滑切换。

## Base URL

`https://api.jingxing.uk/v1`

## Authorization

使用 Bearer Token：

```http
Authorization: Bearer <YOUR_TOKEN>
```

## 最小示例

```bash
curl https://api.jingxing.uk/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <YOUR_TOKEN>" \
  -d '{
    "model": "gpt-5.4",
    "messages": [
      {"role": "user", "content": "你好，介绍一下井陉"}
    ]
  }'
```
