# 快速开始

## 目标

在 5 分钟内完成以下动作：

1. 登录井陉控制台
2. 创建一个可用的 API 令牌
3. 获取基础调用地址
4. 用 `curl` 发出第一条请求

## 你会用到

- 控制台地址：`https://jingxing.uk`
- API 地址：`https://api.jingxing.uk/v1`
- 令牌：在控制台创建

## 首次使用流程

1. 登录控制台并进入令牌管理页面。
2. 创建一个新令牌，选择分组与额度策略。
3. 复制令牌，并保存到你自己的环境变量。
4. 按照[API 快速开始](/docs/getting-started/api-quickstart)发送第一个请求。
