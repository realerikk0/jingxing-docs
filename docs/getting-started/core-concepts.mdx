# 核心概念

## 模型

模型是最终被调用的推理能力，例如 `gpt-5.4`、`claude-sonnet-4-6`、`gemini-3-flash-preview`。

## 分组

分组是能力、价格、供应链与可用模型范围的组合单位。通常用来表达不同档位或线路。

## 令牌

令牌是给客户端和程序使用的访问凭证，可以绑定分组、额度、模型限制或 IP 限制。

## 额度

额度是平台内部的计费单位，用来统一不同模型与不同供应商的消耗。

## 价格

价格决定额度与人民币或美元之间的换算关系，需要跟控制台和充值页保持一致。
