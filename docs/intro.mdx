# Jingxing Docs

井陉是一个统一制式的 AI 网关与工作台，面向 API 调用、模型分组、令牌管理、计费与开发工具集成。

## 你可以从这里开始

- 新用户：先看[快速开始](/docs/getting-started/quickstart)
- API 接入：先看[API 快速开始](/docs/getting-started/api-quickstart)
- 控制台使用：先看[模型分组](/docs/platform/model-groups)与[令牌管理](/docs/platform/token-management)
- 第三方客户端：先看[Claude Code 接入](/docs/integrations/claude-code)

## 文档地图

- `指南`：面向产品使用、配置、计费、错误处理
- `API 参考`：面向 OpenAI 兼容接口与控制台接口
- `开发工具`：面向 Claude Code、Cursor、Cherry Studio 等客户端
- `FAQs`：面向上线前后的高频问题
