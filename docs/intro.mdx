# 井陉doc

专线接入主流模型，无需科学上网。  
井陉的目标很直接：**更快接入，更稳调用，更低迁移门槛。**

## 🧭 井陉是什么

- “井陉”取自河北井陉。陉，指山脉中断形成的通道；井陉是太行八陉之一，自古连接山西高原与华北平原。
- 井陉也留下了“背水一战”的典故。韩信出井陉、背水列阵，以险地破局；井陉因此有了“在狭路中打开通道”的意味。
- 今天的井陉，致力于**模型同权**。最好的模型不该只属于少数人，我们希望大家都能用上最新、最好、最有生产力的 LLM。

## ✨ 为什么选择井陉

- 🌏 不需要翻山越岭，答案就在眼前
- 🔐 安全、可靠，不用担心封号，不需要 KYC
- 🚀 井陉科技，覆盖全球

## 🚀 你会得到

- 无需申请，直接调用最新的 LLM
- 仅需一个 API Key，即可调用全部 LLM
- 团队、环境、客户端流量按分组和令牌隔离
- 成本、调用、消耗与审计集中查看

## 🎯 适合谁

- 想快速接入多个模型的开发者
- 需要管理分组、令牌和额度的团队管理员
- 想把 Cursor、Cherry Studio、Lobe Chat、Claude Code 等工具统一接到一处的人

## 📌 基本信息

- 控制台地址：`https://jingxing.uk`
- API 基础地址：`https://api.jingxing.uk/v1`
- 默认接入方式：OpenAI 兼容接口
- 令牌形式：Bearer Token
- 能力范围：聊天、图片，以及控制台管理接口
