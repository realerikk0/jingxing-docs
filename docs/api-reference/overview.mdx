# API 概览

井陉当前以 OpenAI 兼容调用方式为主，同时保留控制台管理接口。

## 这一栏怎么读

- `API 使用说明`：先看鉴权、Base URL、模型兼容策略
- `中转 API`：当前规范文件已同步到仓库的 `static/openapi/relay.json`
- `控制台 API`：当前规范文件已同步到仓库的 `static/openapi/api.json`

## 推荐补充

- 接入 OpenAPI 渲染插件，把规范转换成站内 API Reference
- API 版本策略
- 稳定接口与实验接口的区分方式
- 限流与幂等性说明
